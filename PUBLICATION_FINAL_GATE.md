# Final Publication Gate

This repository is a public documentation candidate.

A public file may be released only when it has scope, author, rights holder, licence, source, path, checksum when applicable, access layer and review status.

The public chain is: internal working archive, canonical forge, tagged release, DOI deposit, software archive identifier, citation metadata update.

No personal data, private assets, unlicensed images, raw testimony, restricted methods or confidential material may be included in a public release.

Status: candidate pending legal, rights and release review.
