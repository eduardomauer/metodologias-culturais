# Release Immutability Policy

A release is valid only when it is tagged, checksummed and accompanied by citation and metadata files.

## Required before a final release

- README.md
- LICENSE
- CITATION.cff
- codemeta.json
- Zenodo metadata
- publication gate
- metadata schema
- legal gate
- RGPD gate
- checksum manifest

## Rule

Do not rewrite a released tag. If a correction is needed, create a new version.

Status: active for future releases.
