# RGPD, acessibilidade e interoperabilidade

## RGPD

A camada pública só deve expor conteúdos com consentimento, licença e validação editorial. Dados pessoais, imagens identificáveis, relatos sensíveis, localização privada e contactos ficam fora do endpoint público.

## Acessibilidade

A interface pública deve seguir WCAG 2.2 como referência técnica e linguagem clara inspirada na ISO 24495-1:2023. Não declarar conformidade total sem auditoria por critérios.

## Interoperabilidade

Formatos recomendados:

- GeoJSON RFC 7946 para mapa.
- JSON-LD 1.1 para metadados semânticos quando houver maturidade.
- OpenAPI 3.1.1 para endpoints.
- IIIF para património visual quando houver direitos e servidor adequado.
- W3C Web Annotation para anotações curatoriais.
- DataCite/Zenodo para outputs de investigação publicáveis.
