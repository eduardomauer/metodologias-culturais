# MatrizPCI Crosswalk

MatrizPCI is treated as a semantic and documentary crosswalk target, not as an automated API, until technical documentation and terms are verified.

## Current rule

- do not claim direct API integration;
- map fields semantically first;
- keep cultural objects and community-sensitive material out of public repositories until rights and consent are validated;
- publish metadata only when public-safe.

## Candidate fields

- item identifier;
- title;
- territorial reference;
- cultural domain;
- source;
- rights status;
- access layer;
- validation status.

Status: crosswalk only.
