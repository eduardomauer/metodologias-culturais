# Gate público PT/EU — eduardomauer/metodologias-culturais

## Estado

Este repositório é tratado como camada pública controlada da operação MILK / Atlas.

## Papel

- Repositório: `eduardomauer/metodologias-culturais`
- Função: camada pública metodológica principal
- Estado: candidato canónico público, sujeito a validação humana antes de release final.

## Bloqueios obrigatórios

Até validação formal, permanecem bloqueados:

- release final;
- DOI Zenodo;
- envio para Software Heritage;
- publicação de datasets sensíveis;
- publicação de materiais com direitos não confirmados;
- publicação de dados pessoais, testemunhos identificáveis, menores, contactos, localizações sensíveis ou imagens sem autorização.

## Regras de publicação

Só pode entrar na camada pública material que esteja classificado como público, licenciado, minimizado, sem segredos técnicos, sem dados pessoais indevidos e com autoria ou titularidade documentada.

Em caso de dúvida, o material fica restrito ou confidencial até revisão.

## Repositório bloqueado externo

`eduardomauer/runner` não é fonte oficial desta camada. Deve permanecer tratado como bloqueado / não-canónico.
